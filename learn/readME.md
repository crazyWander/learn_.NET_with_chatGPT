Конечно, вот несколько более сложных задач на C# для улучшения навыков:

Создайте консольную игру "Виселица". Программа должна загадывать случайное слово и предлагать пользователю угадывать буквы. Пользователь должен иметь ограниченное количество попыток. Если он не угадывает слово за отведенное количество попыток, он проигрывает.

Создайте простой текстовый редактор на C#. Редактор должен позволять открывать, редактировать и сохранять текстовые файлы в различных форматах.

Реализуйте приложение для хранения задач (TODO-лист). Приложение должно позволять пользователю создавать, редактировать и удалять задачи, а также отмечать их как выполненные.

Реализуйте игру "Сапер". Программа должна создавать поле мин размером NxM, где N и M задаются пользователем. Затем программа должна позволять пользователю открывать ячейки и отмечать их флажками. Если пользователь открывает ячейку с миной, он проигрывает.

Создайте консольное приложение, которое позволяет пользователю конвертировать валюты. Приложение должно получать курсы валют из открытого источника и позволять пользователю конвертировать деньги между различными валютами.

Надеюсь, эти задачи помогут вам улучшить навыки программирования на C#.
